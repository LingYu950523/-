# 函数对象

* 具名函数
function 函数名（形式参数1， 形式参数2）{
  语句
  return 返回值
}

* 匿名函数
上面的具名函数，去掉函数名就是匿名函数
let a = function(x, y){return x+y}
这种形式也被叫做函数表达式
如果是let a = function fn(x, y){return x+y}，则fn的作用域只在这句话中，其他地方不存在fn
